```bash
# 전체 테스트 실행
pnpm test

# 단위 테스트만 실행
pnpm test:unit

# E2E 테스트만 실행  
pnpm test:e2e

# 커버리지 포함 테스트
pnpm test:coverage

# 감시 모드 테스트
pnpm test:watch
```

1. **개발 시**: `pnpm test:watch` (실시간 테스트)
